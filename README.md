# C Programming Beginner Tutorial

This repository is made for absolute beginners who want to learn C programming step by step.

## Topics Covered
- First Word
- Varriables
- Input / Output
- If & If-Else
- Loops
- Functions
- Arrays
- Pointers
- Structures

## Folder Structure
Each folder contains simple C programs with comments for easy understanding.

## How to Run Programs
1. Install GCC compiler
2. Open terminal in program folder
3. Compile:
   gcc filename.c -o output
4. Run:
   ./output
